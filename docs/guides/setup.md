# Setup Guide

## First-Time Setup

Copy environment template:

```bash
cp .env.example .env
```

Then update values in `.env` as needed for your local environment.

## Install

```bash
npm install
```

## Run

```bash
npm run dev
```

## Environment

Minimum local value:

```env
VITE_API_URL=http://localhost:3000
```
